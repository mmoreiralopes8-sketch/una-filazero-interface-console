O Fila Zero é uma aplicação de console simples desenvolvida em C# para demonstrar um fluxo básico de autenticação de usuários. O foco do projeto é a organização de código utilizando métodos e manipulação de entrada/saída no terminal.

📋 Funcionalidades
Interface Customizada: Cabeçalho estilizado no console.

Codificação UTF-8: Suporte para acentos e caracteres especiais.

Simulação de Autenticação: Validação de credenciais pré-definidas.

Limpeza de Tela: Gerenciamento da visualização para uma melhor experiência do usuário.
